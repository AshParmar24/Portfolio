# Equity Sentiment NLP

Transforming financial news into sentiment-based market signals for equity analysis.

## Overview

**Domain:** Finance / NLP

**Business Value:** Converts unstructured financial news into structured investment insight.

## Key Skills

- NLP
- Sentiment Analysis
- Market Signals
- Text Processing
- Backtesting

## Project Highlights

- Processed financial headlines into sentiment features
- Created sentiment-based trading signals
- Discussed look-ahead bias and market validation
- Connected NLP output to investment decision support
