# Customer Churn Prediction

Predicting customer churn using leakage-safe machine learning, cross-validation and SHAP explainability.

## Overview

**Domain:** Customer Analytics / Machine Learning

**Business Value:** Identifies customers most likely to leave and supports targeted retention campaigns.

## Key Skills

- Classification
- Feature Engineering
- Cross-Validation
- SHAP
- Business Recommendations

## Project Highlights

- Benchmarked 12 machine learning classifiers
- Used F1 Score and AUC-ROC to evaluate imbalanced classification performance
- Applied one-hot encoding for categorical features
- Performed feature selection after the train/test split to avoid leakage
- Used SHAP to explain churn drivers
