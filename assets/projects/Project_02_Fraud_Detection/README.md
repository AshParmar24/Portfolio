# Financial Fraud Detection

Detecting suspicious transaction behaviour using machine learning and graph-based analytics.

## Overview

**Domain:** Finance / Fraud Analytics

**Business Value:** Supports early identification of suspicious transactions and high-risk account behaviour.

## Key Skills

- Fraud Detection
- Graph Analytics
- NetworkX
- Imbalanced Classification
- Risk Scoring

## Project Highlights

- Modelled fraud as an imbalanced classification problem
- Used graph-based transaction relationships to identify suspicious patterns
- Prioritised recall and F1 Score due to the cost of missed fraud
- Produced risk-focused business recommendations
