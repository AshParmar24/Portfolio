# Renewable Asset Dashboard

Monitoring renewable asset performance using dashboard analytics and anomaly detection.

## Overview

**Domain:** Energy / Operations Analytics

**Business Value:** Helps identify underperforming wind and solar assets for operational review.

## Key Skills

- Dashboarding
- Anomaly Detection
- Renewable Energy
- Asset Analytics
- Data Visualisation

## Project Highlights

- Created asset-level performance monitoring
- Used anomaly detection to flag underperformance
- Calculated commercially relevant metrics such as capacity factor
- Presented results in a dashboard-style format
