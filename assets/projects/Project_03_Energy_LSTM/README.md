# Energy Demand Forecasting

Forecasting short-term electricity demand using LSTM neural networks and time-series features.

## Overview

**Domain:** Energy / Forecasting

**Business Value:** Supports operational planning, grid balancing and demand management.

## Key Skills

- LSTM
- Time-Series Forecasting
- Deep Learning
- Energy Analytics
- Sequence Modelling

## Project Highlights

- Built a neural network forecasting workflow
- Captured sequential demand patterns using LSTM architecture
- Included calendar and seasonality effects
- Discussed operational forecasting use cases
